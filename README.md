# Conference Website

This is a static conference website built with HTML, CSS, and vanilla JavaScript.

## Project Structure

- **index.html**: Main landing page
- **pages/**: Contains additional pages for schedule, speakers, venue, and registration
- **css/**: Contains styles for the website
- **js/**: Contains main JavaScript logic and data files
- **robots.txt**: Manages web crawler access

## Setup Instructions

1. Clone the repository or download the project files.
2. Open the `index.html` file in your web browser to view the conference website.
3. Navigate through the different pages using the navigation bar.

## Features

- Responsive design for mobile and desktop views.
- Dynamic loading of schedule and speakers data using JavaScript.
- Simple registration form for attendees.

## Technologies Used

- HTML
- CSS
- JavaScript (vanilla)
- JSON for data storage
